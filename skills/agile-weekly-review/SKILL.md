# Agile Weekly Review

## Goal
Deliver a clear weekly review based on goals and measurable outcomes.

## Steps
1. Read `memory/goals.md`.
2. Use available tools to fetch current KPI values.
3. Compare current vs target with numeric gaps.
4. Summarize what worked and what failed.
5. Propose exactly 3 actions for next week.

## Output Format
- KPI status table
- Wins
- Misses
- Next 3 actions with owners and timing
