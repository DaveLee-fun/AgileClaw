# KPI Daily Check

## Goal
Check daily KPI movement and report actionable deltas.

## Steps
1. Read today's goals and KPI baselines from `memory/goals.md`.
2. Measure KPI values with available tools or APIs.
3. Compute day-over-day delta.
4. Highlight any risk KPI that is off-track.
5. Suggest immediate next action for today.

## Output Format
- Today snapshot
- Delta vs yesterday
- Risks
- Today's immediate action
