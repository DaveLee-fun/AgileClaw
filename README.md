# AgileClaw

Version: `0.1.0`

AgileClaw is a KPI-driven autonomous agent project.

This repository currently uses `src/` as the executable code root.

## Start

```bash
cd src
cp config.example.yaml config.yaml
pip install -r requirements.txt
playwright install chromium
python main.py
```
